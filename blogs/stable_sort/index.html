stable_sort() #

  • It comes under the <algorithm> header file
  • stable_sort is used to sort the elements from [first, last) in ascending order.
  • stable_sort is like std::sort, but stable_sort() keeps the relative order of elements with equivalent values.
  • Example of comparing between stable_sort() and sort()

"kid","use","ask","ten","lie","yes","bad","all","car","boy","why","hit","one","our","add","guy","few","bar","off","low","day","per","war","bit","can","gun","you","six","his","job","act","bed","sex","fly","bag","own","arm","put","out","any","hot","lay","eye","cut","now","nor","and","oil","may","her","big","eat","dog","leg","air","say","age","red","not","its","let","the","man","lot","for","get","she","who","new","way","sit","buy","box","ago","law","key","end","see","old","sea","far","him","but","art","how","pay","win","set","top","try","cup","run","yet","tax","gas","son","too","two","die"

stable_sort(words.begin(), words.end(),[](const string &a, const string & b){
    return a.length() < b.length(); 
});
